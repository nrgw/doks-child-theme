* 참가자격
  * 학부 및 대학원 재학생
  * 이학 및 공학 계열 박사학위 소지자는 신청 불가
* 팀구성
  * 2인 이하 x 17팀
  * 신청팀이 많을 경우 심사를 통해 참가팀 선발
* 출제문항
  * 1번: 중력파 데이터 분석 관련 코드 작성 문제
  * 2번: 수치상대론 관련 코드 작성 문제
* 사용가능 언어
  * GCC (C / C++ / Fortran)
  * Python
* 채점
  * 각 문제의 배점은 동일
  * 채점기준은 문제별로 상이
* 경진대회 수상
  * 1등팀: iPad mini x 팀원수
  * 2등팀: 애플 워치 SE, 갤럭시 워치 중 택1 x 팀원수
